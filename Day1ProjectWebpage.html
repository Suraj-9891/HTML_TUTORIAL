HTML (HyperText Markup Language)

HTML is the standard markup language used to create web pages. It provides the structure and content of a web page, which is then styled using CSS and made interactive using JavaScript.

What does HTML do?

  1. Defines the structure and organization of web page content
  2. Adds meaning to content through semantic elements
  3. Provides a way to link documents and create hyperlinks
  4. Enables the creation of forms, tables, and other interactive elements
  5. Supports multimedia content like images, audio, and video

Basic HTML Concepts

  1. Elements: Represented by tags (e.g., p, img, a)
  2. Tags: Surround content with angle brackets (< and >)
  3. Attributes: Provide additional information about elements (e.g., id, class, src)
  4. Values: Assigned to attributes (e.g., id="header", class="nav-link")

Basic HTML Structure

Types of HTML Elements

  1. Block-level elements: Occupy a rectangular block of space (e.g., p, h1, div)
  2. Inline elements: Fit within surrounding text or other inline elements (e.g., span, a, img)
  3. Empty elements: Do not have content (e.g., br, hr, img)

HTML Attributes

Attributes provide additional information about an element. They are added to the opening tag and usually consist of a name and value

  • id - Unique identifier for the element
  • class - Class name for styling and JavaScript selection
  • style - Inline CSS styles
  • title - Tooltip text
  • alt - Alternative text for images
  • src - Source URL for images and media
  • href - Hyperlink URL
  • target - Link target (e.g., _blank, _self)

HTML Versions

  1. HTML 1.0 (1993): First version of HTML
  2. HTML 2.0 (1995): Added features like forms and tables
  3. HTML 3.2 (1997): Introduced stylesheets and scripting
  4. HTML 4.01 (1999): Improved accessibility and internationalization
  5. HTML5 (2014): Added multimedia support, canvas, and web storage

Why Use HTML?

  • Platform independence: HTML works on multiple devices and browsers
  • Easy to learn: Basic HTML concepts are simple to grasp
  • Flexible: HTML can be used for web pages, email, and mobile apps
  • Accessible: HTML provides features for accessibility and semantic meaning

Real-World Applications of HTML

  1. Web development: Building websites and web applications
  2. Email marketing: Creating HTML emails for marketing campaigns
  3. Mobile app development: Using HTML for hybrid mobile apps
  4. Document creation: Creating documents with HTML for printing or digital distribution
